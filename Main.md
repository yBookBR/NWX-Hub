# NWX-Hub
Hi! This script was made 100% by me, Book. The script is intended for the game “Baile do Urso” on the Roblox platform. Feel free to comment your opinion about the script’s structure.

`Version 1.1`

## Features
This script allows you to switch to any team in the experience and includes features like RGB Glow, a dot above players' heads, and a name displayed over their heads. It also adds functions such as viewing your FPS in real time, the server ping and server ID, as well as allowing you to modify the player's speed and run the Infinite Yield script.
